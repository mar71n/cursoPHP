CSS -> 
	La idea de CSS es seleccionar determinados elementos y aplicarle ciertas reglas
	
	selector por tag		p {}  afecta a todos los <p>
	selector por clase		.noticia {}	  afecta a cualquier elemento class='noticia'
	selector por id			#contenedor {}	afecta al elemento id='contenedor'
	combinar				p.noticia {} 
	agrupar					p, .noticia, #ventana {}
	herencia				tr td {} todos los <td> que estén dentro de un <tr>
	herencia				#contenedor td { }
